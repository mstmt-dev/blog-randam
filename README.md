# blog-randam
テーマを絞らない雑多ブログ。 

# 背景と概要
最近ふと思ったんだけど、ブログってどこに書くのが一番気楽なんだろう？
自前で WordPress サイトを立ててみるのも、それはそれで面白いんだけど、ドメインの管理とかプラグイン更新とか、やる前から何となく面倒が頭をよぎる。
Qiita みたいなサービスに投稿するのも便利だけど、サービスごとの文化とかフォーマットに合わせる必要があって、気軽にひとこと書くにはちょっと肩に力が入る。
ということで、「じゃあもう GitHub に書けばいいのでは？」という気持ちになった。
普段から触っているし、アカウント管理もシンプルだし、Markdown で書けるし、履歴がそのまま残るのもむしろ味があっていい。ブログというよりは自分のノートの延長みたいに使えるのも気に入ってる。
というわけで、ここを雑多ブログの場所にしてみることにする。
特にテーマも決めず、思いついたことをそのまま書いていく感じのゆるいスタイルで。
技術のことも書くかもしれないし、日々の気づきやちょっとしたメモも書くかもしれないし、逆に何も書かない日もあると思う。
でもそのくらいのラフさが今の自分にはちょうどいい気がする。
どこまで続くかはわからないけど、まあ、気楽にいこう。
